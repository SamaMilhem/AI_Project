How to Play Mastermind

The computer will generate a secret code of 4 colors. Your goal is to guess the correct colors and their positions.

After each guess, you'll receive feedback:

  • Black peg:
    • Correct color in the correct position
  • White peg:
    • Correct color in the wrong position

Notice: You have only 10 attempts to guess the code, so use hints wisely!

Game Modes:

  • The Classic Game: You can choose to play alone, where you try to guess the computer's secret code.
  • Play Against a Genetic Agent: Challenge the computer, which uses a Genetic Algorithm to guess the secret code. Try to outsmart it!
  • Q-Learning Agent: You can see a run of a Q-Learning agent on a secret code that you select. Watch how the agent learns to guess your code.

Psst... You will earn coins for each winning game against the genetic agent. Challenge yourself with different difficulties by changing the number of optional colors when playing the classic game or against the genetic agent !
