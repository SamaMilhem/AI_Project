How to Play Mastermind

The computer will generate a secret code of 4 colors. Your goal is to guess the correct colors and their positions.

After each guess, you'll receive feedback:

  • Black peg:
    • Correct color in the correct position
  • White peg:
    • Correct color in the wrong position

Notice: You have only 10 attempts to guess the code. Therefore, use hints wisely! psst... You will earn coins for each winning game against the agent.

Challenge yourself with different difficulties by changing the number of optional colors!
